📋 MURAL DE AVISOS

Clique em + Novo Evento para cadastrar um aviso na agenda da equipe.

Cada aviso tem uma data do evento — quando ele acontece — e uma data de exibição opcional — até quando aparece no mural.

Sem data de exibição, o aviso some automaticamente quando o evento começa. Com ela, continua visível até o prazo definido.

🎂 Os aniversários do mês aparecem aqui para facilitar a criação de um aviso de parabéns.

Avisos podem ser para toda a empresa ou segmentados por facção — jogadores veem apenas o que é relevante para eles.
